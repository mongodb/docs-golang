===============
Find a Document
===============

.. default-domain:: mongodb

You can retrieve a single document from a collection with the
``FindOne()`` method.

The following example passes a query filter to the ``FindOne()`` method,
which matches documents in the ``movies`` collection where the value
of the ``title`` field is *The Room*, returning the first document found.

Expected Result
---------------

After you run the code example, you will see output similar to:

.. code-block:: json
   :copyable: False

   {
     ...
     "title": "The Room",
     "year": 2003,
     ...
   }
