=================
Insert a Document
=================

.. default-domain:: mongodb

You can insert a single document in a collection by using the ``InsertOne()``
method.

The following example creates a new document containing a ``title`` and ``text``
field nad inserts it into the ``Haikus`` collection of the ``insertDB`` database:

Expected Result
---------------

After you run the code example, a new document with an ``_id`` that matches
the output will be present:

.. code-block:: json
   :copyable: false

   {
     _id: ObjectId("..."),
     title: 'An Insert Haiku',
     text: 'No bytes, no problem. Inserting a document. In MongoDB'
   }
