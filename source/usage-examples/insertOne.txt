=================
Insert a Document
=================

.. default-domain:: mongodb

You can insert a single document in a collection by using the ``InsertOne()``
method.

The following example creates a new document containing a ``title`` and ``text``
field nad inserts it into the ``Haikus`` collection of the ``insertDB`` database:

Expected Result
---------------

After you run the code example, the document will include the
``avg_rating`` field with the specified value:

.. code-block:: json
   :copyable: false

   // result truncated
   {
      "_id" : ObjectId("5eb3d668b31de5d588f42a7a"),
      ...
      "name" : "Green House Cafe",
      "restaurant_id" : "40372112",
      "avg_rating" : 4.4
   }
